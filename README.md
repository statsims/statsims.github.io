# Stats Sims

Interactive simulations for learning applied statistics and quantitative reasoning.

🔗 Live site: https://statsims.github.io

---

## Overview

**Stats Sims** is a collection of browser-based, interactive tools designed to support conceptual understanding of key statistical ideas. These simulations emphasize interpretation, visualization, and intuition over computation.

The tools are intended for use in applied quantitative courses, particularly for students who are developing skills as consumers of research rather than statisticians.

---

## Available Simulations

### Correlation Explorer
Explore how direction, strength, noise, and outliers affect Pearson’s correlation.

### Confidence Interval Explorer
Visualize how confidence intervals behave across repeated samples and how coverage relates to the confidence level.

### Sampling Distribution Explorer
Observe how sample means form a distribution and how sample size affects variability (standard error).

### Significance via Confidence Intervals
Connect confidence intervals to hypothesis testing and understand when results are inconsistent with a null value.

### DAG Explorer
Interact with causal diagrams to understand confounding, mediation, colliders, and adjustment strategies.

### Effect Size Explorer
Interpret practical significance and understand how magnitude differs from statistical significance.

### Regression Explorer (OLS)
Explore linear relationships, fitted lines, and prediction in regression models.

### Unlikely Outcomes Simulator
Build intuition for probability, rare events, and what counts as “unusual.”

---

## Purpose

These tools are designed to:

- Reinforce conceptual understanding of statistical ideas  
- Support visual and interactive learning  
- Reduce reliance on formula memorization  
- Bridge theory and interpretation in applied contexts  
- Provide reusable teaching demonstrations  

---

## How to Use

1. Visit the live site: https://statsims.github.io  
2. Select a simulation from the homepage  
3. Adjust controls and observe how outputs change  
4. Focus on interpretation rather than calculation  

---

## Technical Notes

- Built with HTML, CSS, and JavaScript  
- Uses Plotly.js for interactive visualizations  
- Fully client-side (no backend required)  
- Hosted via GitHub Pages
