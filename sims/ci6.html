Help with Interpretation

How to use this app: Start with a fixed population and confidence level, then click Redraw intervals several times. Watch how the intervals change from sample to sample even though the population mean stays fixed.

Main purpose: This app shows that a confidence interval is not a statement about certainty for one specific interval. It is about the long-run performance of a procedure across repeated samples.

What to look for: Green intervals capture the true mean. Red intervals miss it. The observed coverage should drift toward the nominal level as the number of intervals increases.

Key comparison: Increase n and redraw. The intervals should usually become narrower. Increase the confidence level and redraw. The intervals should usually become wider.

Interesting case: Use a skewed population with a small sample size. Coverage can depart from the nominal level more than students expect. This is a useful reminder that interval performance depends on assumptions and sample size.

Common aha moments: Even a 95% procedure still produces some misses. Wider intervals are not automatically better; they trade precision for coverage. Changing what you display does not change the underlying procedure.

Visual cues: The black vertical line marks the true population mean. The optional gray band is only a guide for the eye. It is not part of the interval calculation.

Warning: Coverage can be off when the population is strongly skewed and n is small.
