Help with Interpretation

Main idea: A confidence interval gives a range of parameter values that remain plausible under the chosen confidence level and method.

Significance link: If the null value lies outside the interval, the observed result is inconsistent with that null at the matching level. If the null value lies inside the interval, the data are more compatible with that null.

What to look for: Move the null-value slider. Watch what happens when the null enters or leaves the confidence interval. That transition is the visual link to significance.

Key comparison: Increase the sample size and the interval usually narrows. Increase the confidence level and the interval usually widens.

Important caution: “Outside the interval” is not a complete substantive interpretation. It only addresses compatibility with the null value, not the practical importance of the effect.
